# Knowledge-Intelligence-Application---RAG-Based-Agentic-AI


This project is a Retrieval-Augmented Generation (RAG) based AI application built using Python, LangChain, OpenAI, and ChromaDB.

The application allows users to ask questions based on uploaded knowledge/documents.  
Instead of relying only on the AI model’s training data, the system retrieves relevant information from stored documents and uses it to generate accurate responses.

---

## Features

- Conversational AI chatbot
- Retrieval-Augmented Generation (RAG)
- Vector database using ChromaDB
- LangChain integration
- OpenAI GPT integration
- Conversation memory support
- Document-based question answering

---

## Tech Stack

- Python 3.11
- LangChain
- OpenAI API
- ChromaDB
- Flask / FastAPI
- HuggingFace Embeddings
